# Online Examination System

## Overview

The Online Examination System is a Java-based console application designed to conduct multiple-choice examinations. It provides secure login functionality, profile management, password updates, timer-based examinations, automatic submission, and result generation.

## Features

* User Login Authentication
* Update Profile
* Change Password
* Multiple Choice Questions (MCQs)
* Timer-Based Examination
* Auto Submission
* Result Calculation
* Logout Functionality

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Arrays
* Scanner Class
* Timer Logic

## How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Compile and run `OnlineExaminationSystem.java`.
4. Login using valid credentials and start the examination.

## Learning Outcomes

* Authentication Systems
* Java Collections and Arrays
* Timer Management
* Console-Based Application Development
* Examination Workflow Design
